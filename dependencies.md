# AICare+ Dependencies

This document lists all the Python dependencies required to run the AICare+ application.

## Core Dependencies

- flask==2.3.3
- flask-sqlalchemy==3.1.1
- gunicorn==23.0.0
- ipython==8.18.1
- pandas==2.1.4
- pyautogen==0.2.17
- ollama==0.1.6
- trafilatura==1.6.3
- sentence-transformers==2.3.1
- openai==1.3.8
- chromadb==0.4.22
- email-validator==2.1.0.post1
- psycopg2-binary==2.9.9

## Installation

To install all dependencies, run:

```bash
pip install flask flask-sqlalchemy gunicorn ipython pandas pyautogen ollama trafilatura sentence-transformers openai chromadb email-validator psycopg2-binary
```

Or if you prefer to install specific versions:

```bash
pip install flask==2.3.3 flask-sqlalchemy==3.1.1 gunicorn==23.0.0 ipython==8.18.1 pandas==2.1.4 pyautogen==0.2.17 ollama==0.1.6 trafilatura==1.6.3 sentence-transformers==2.3.1 openai==1.3.8 chromadb==0.4.22 email-validator==2.1.0.post1 psycopg2-binary==2.9.9
```
